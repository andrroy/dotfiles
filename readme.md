## How

1. Install iTerm2

2. Install Command line tools

```xcode-select --install```

3. Clone repo to home directory

```git clone https://github.com/andrroy/dotfiles.git```

4. Run install script

```cat install.sh | sh```

5. Apply iTerm theme

6. Restart iTerm
